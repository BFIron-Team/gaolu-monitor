\# 高炉炼铁工艺智能监控系统



\*\*项目简介\*\*：

本项目旨在为高炉炼铁工艺打造一个直观、专业的可视化仪表盘系统，打通“算法可用”到“现场好用”的最后一公里。



\*\*团队成员\*\*：

\- 赵梓婷（项目主持）：总体架构设计、三线可视化模块开发

\- 王盈盈（成员）：数据模拟脚本开发、预警逻辑实现

\- 李秀萍（成员）：前端布局、仪表盘组件开发、系统测试



\*\*技术栈\*\*：

\- Python + Streamlit + Plotly

\- Pandas + NumPy（数据模拟）
