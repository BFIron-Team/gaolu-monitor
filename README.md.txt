炼铁工艺关键控制指标的时空演变与交互系统
📖 项目简介
本项目旨在为一线操作人员提供一个直观的工业风格仪表盘，将后台复杂的算法分析结果转化为可视化的图表和预警信息，解决“算法可用但现场难用”的痛点。
📁 项目结构
bf_ironmaking/
├── app.py # Streamlit 主程序入口
├── plotly_test.py # Plotly 绘图测试文件
└── README.md # 项目说明文档
✨ 核心功能模块
“三线对比图”：在同一张折线图上对比显示“预测曲线”、“历史曲线”和“实时曲线”
“关键指标仪表盘”：展示风温、炉压等核心数据
“决策建议预警”：高亮显示操作建议
🛠 技术栈
“前端框架”：Streamlit
“可视化库"：Plotly
*   **数据处理**：Pandas
*   **版本控制**：Git & GitHub
